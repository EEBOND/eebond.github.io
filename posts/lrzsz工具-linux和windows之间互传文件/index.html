lrzsz工具 Linux和Windows之间互传文件

作者 收录于 Linux
 2021-11-16  约 131 字   预计阅读 1 分钟 
目录
    • 安装lrzsz工具
    • Linux端接收来自Windows端的文件
    • Linux端发送文件到windows端

该工具需要在支持文件传输的终端工具中使用，如xshell可以，而putty不能。

安装lrzsz工具

yum install -y lrzsz

Linux端接收来自Windows端的文件

rz

输入命令后会弹出文件选择框

Linux端发送文件到windows端

sz [文件路径]

输入命令后会弹出文件保存位置选择框
